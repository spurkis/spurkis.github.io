TAP::Harness HTML plugin

TAP::Formatter::HTML is a plugin for the test automation framework, TAP::Harness. It was written to highlight visually where & why your test suite is breaking. Here are some examples:

  • test-output.html
  • DBD-SQLite-example.html
  • Template-example.html

The code is freely available: /spurkis/TAP-Formatter-HTML
